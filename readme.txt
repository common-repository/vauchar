The official plugin by Vauchar to integrate the Vauchar API with the woocommerce

== Description ==

This is the long description.  No limit, and you can use Markdown (as well as in the following sections).

Now you can easily use the Vauchar API with woocommerce using this plugin.
Just install the plugin and get started, enter your credetials and you are ready to get started.
To know more please http://vauchar.com

== Installation ==

1. Make sure woocommerce plugin is installed and activated.
2. Make sure coupons are enabled by going to Settings->Woocommerce->Settings->Checkout.
3. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
4. Activate the plugin through the 'Plugins' screen in WordPress
5. Use the Settings->Vauchar screen to enter your Vauchar API credentials and save.
